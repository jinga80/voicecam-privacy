보이스캠(VoiceCam) 개인정보처리방침

시행일: 2026-07-03

보이스캠(이하 "앱")은 코리아세일즈가 개발·운영하는 앱으로, 사용자가 자신의 일상과 통화를 녹음하고, 본인의 클라우드(구글 드라이브)나 외장 저장장치에 백업하도록 돕는 도구입니다. 본 방침은 앱이 어떤 정보를 다루고, 어떻게 보호하는지 설명합니다.

본 방침은 코리아세일즈가 제공하는 보이스캠 패밀리 — 장사노트·회의노트·수업노트·세일즈노트·건강노트·육아노트·부동산노트·상담노트·인터뷰노트·말씀노트 등 동일 엔진 기반의 업종별 버전에도 동일하게 적용됩니다. 모든 버전은 녹음·정리를 기기 내에서 처리하며 개발사 서버에 녹음을 전송하지 않습니다.

1. 한눈에 보기

  • 앱은 사용자 본인의 녹음 파일만 다룹니다. 개발사는 사용자의 녹음 내용을 수집·열람·전송하지 않습니다.
  • 녹음 파일은 사용자가 직접 연결한 본인의 구글 드라이브 계정 또는 사용자가 선택한 외장 폴더에만 저장됩니다.
  • 개발사 서버는 없으며, 녹음이 개발사나 제3자에게 전송되지 않습니다.

2. 수집·처리하는 정보

앱은 다음을 기기 내에서 처리합니다. 개발사로 전송되지 않습니다.

  • 음성 녹음 파일: 사용자가 만든 .m4a 파일.
  • 통화 녹음 파일(선택 기능): 사용자가 "통화 녹음 자동 백업"을 켜고 휴대폰의 통화 녹음 폴더를 직접 선택한 경우, 그 폴더의 파일만 읽어 백업합니다.
  • 설정 값: 백업 옵션, 선택한 백업 폴더 위치, 백업 진행 기록 등(기기 내 저장).

앱은 이름·이메일·연락처·위치·광고식별자 등 개인 식별 정보를 수집하지 않습니다.

2-1. 출시 알림 신청 정보 (홈페이지)

  • 홈페이지(voicecam.net 및 업종별 페이지)에서 출시 알림을 신청하실 때 입력한 이메일 또는 전화번호를, 코리아세일즈가 출시 안내 목적으로만 수집·이용합니다.
  • 보유·이용 기간: 출시 안내 완료 또는 수신 거부 요청 시까지. 이후 지체 없이 파기합니다.
  • 수신 거부·삭제: korsales.dev@gmail.com 으로 요청하시면 즉시 처리합니다.

2-2. 홈페이지 이용 정보

홈페이지는 방문 통계를 위해 구글 애널리틱스 등 기본 분석 도구를 사용할 수 있습니다. 이는 익명 통계 목적이며, 브라우저 설정으로 거부할 수 있습니다.

3. 정보의 저장과 전송

  • 구글 드라이브 백업: 사용자가 본인 구글 계정으로 인증하면, 앱은 drive.file 범위(앱이 만든 파일만 접근)로 사용자 본인 드라이브의 '보이스캠' 폴더에 녹음을 업로드합니다. 개발사는 접근할 수 없습니다.
  • 외장(USB/SSD) 백업: 사용자가 선택한 폴더에만 복사됩니다.
  • 업로드는 기기에서 구글 드라이브로 직접 이루어지며, 개발사 서버를 거치지 않습니다.

4. 사용하는 권한과 이유

  • 마이크·마이크 포그라운드 서비스: 녹음 및 화면이 꺼져도 녹음 유지.
  • 폴더 접근(사용자 선택): 사용자가 직접 선택한 통화 녹음 폴더만 읽어 백업합니다(통화 백업 기능을 켤 때만). '모든 파일 접근' 권한은 사용하지 않습니다.
  • 인터넷: 구글 드라이브 업로드.
  • 알림: 녹음 진행 표시.
  • 부팅 완료 수신: '재부팅 후 자동 녹음' 옵션 사용 시 재개 안내.

5. 제3자 제공·공유

  • 개발사는 사용자의 녹음을 제3자에게 제공·판매·공유하지 않습니다.
  • 구글 드라이브는 사용자가 선택한 저장소 제공자이며, 데이터는 사용자 본인 계정에 귀속됩니다(구글 약관·정책 적용).

6. 보관과 삭제

  • 녹음 파일은 앱 목록(개별/선택/일괄 삭제)이나 본인 드라이브·외장 폴더에서 직접 삭제할 수 있습니다.
  • 앱 삭제 시 기기 내 설정과 앱 전용 폴더의 파일이 함께 삭제됩니다. 이미 백업된 파일은 사용자가 별도로 관리합니다.

7. 어린이

본 앱은 만 14세 미만 아동을 대상으로 하지 않습니다.

8. 법적 유의 (대한민국 통신비밀보호법)

  • 본인이 참여한 대화나 본인의 일상 녹음은 일반적으로 적법합니다.
  • 본인이 참여하지 않은 타인 간 대화를 동의 없이 녹음하는 것은 불법입니다. 사용자는 관련 법령을 준수할 책임이 있습니다.

9. 변경

본 방침이 변경되면 앱 또는 본 페이지를 통해 고지합니다.
