녹음부터 백업까지, 한 앱에서

놓치고 싶지 않은 순간, 다시 듣고 싶은 대화, 만약을 위한 기록.

실시간 파형 녹음

한 번 녹음과 하루종일 연속 녹음(30분 자동 분할). 화면이 꺼져도 백그라운드로 이어지고, 소리는 파형으로 눈에 보입니다.

안전한 자동 백업

녹음이 끝나면 내 구글 드라이브에 날짜별로 자동 업로드. USB·SSD 외장에도 동시 백업, Wi-Fi 전용 옵션까지.

통화 녹음 백업

휴대폰의 통화 녹음 폴더를 직접 골라 자동 백업. '모든 파일 접근' 권한 없이, 선택한 폴더만 안전하게.

내 데이터는 내 것

개발사 서버가 없습니다. 녹음은 오직 본인 계정과 외장에만 저장되며, 보관·선택·일괄 삭제도 직접 관리합니다.
