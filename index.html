왜 보이스캠인가
그 순간은, 다시 오지 않습니다

중요한 통화의 약속, 잊어버린 회의의 결론, 매일 밤 나도 모르게 멈추는 숨, 그리고 혼자 맞닥뜨린 위험한 순간. 보이스캠은 기록하고, 정리하고, 분석하고, 지켜줍니다 — 구독료도, 클라우드 비용도 없이.

회의·강의·통화를 녹음하면 보이스캠이 글로 옮기고, 핵심 요약과 할 일, 키워드까지 자동으로 정리합니다. 이 모든 정리가 휴대폰 안에서, 무료로 이뤄져요.

🎙 하루종일 연속 녹음

켜두기만 하면,
온종일 끊김 없이

화면이 꺼져도, 가방 속에서도 멈추지 않습니다.

‘하루종일’ 모드는 한 번 켜두면 24시간 내내 백그라운드로 이어 녹음합니다. 길어진 녹음은 30분마다 자동으로 나눠 저장되고, 소리는 실시간 파형으로 눈에 보여요. 잠깐의 순간도 놓치지 않도록.

  • 하루종일 연속 녹음 · 30분 자동 분할
  • 화면 꺼짐·앱 종료에도 백그라운드 유지
  • 한 번 녹음 모드 · 실시간 파형 · 북마크
  • 재부팅 후 자동 재개 · 빠른 타일/위젯
📞 통화 녹음 자동 백업

통화 녹음, 자동으로
백업되고 있을까요?

스마트폰 통화 녹음 — 대부분 ‘자동 백업’은 안 됩니다.

휴대폰에 쌓인 통화 녹음, 기기를 잃어버리면 그대로 사라집니다. 보이스캠은 통화 녹음 폴더를 직접 골라 새 통화가 생길 때마다 내 드라이브로 자동 백업해요. ‘모든 파일 접근’ 권한 없이, 선택한 폴더만 안전하게.

  • 통화 녹음 폴더 선택 → 새 통화 자동 백업
  • 기존 통화 전체 백업도 한 번에
  • Wi-Fi 전용 옵션 · 업로드 후 정리
🌙 수면 분석

잠든 사이,
몸이 보내는 신호를 읽다

수면 모드로 녹음하면 진폭을 온디바이스로 분석해 코골이 점수와 무호흡 의심 구간을 찾아드립니다. 의심되는 순간은 그 지점부터 바로 들어볼 수 있어요.

  • 코골이 점수 · 무호흡 의심 횟수
  • 밤새 강도 타임라인 + 이벤트 마커
  • 이비인후과·수면클리닉 상담 자료로
🆘 안전 SOS

위급한 순간,
당신 대신 전화를 겁니다

녹음 중 위급한 큰 소리가 감지되면 전체화면 경보가 뜨고, 취소하지 않으면 지정한 연락처로 자동 전화합니다. 혼자 자는 밤에도, 야외에서도.

  • 외침·위험 소리 자동 감지 (온디바이스)
  • 취소 카운트다운 → 자동 발신
  • 민감도 · 연락처 직접 설정
